# ERP Assistant

- The ERP assistant is build on top of langgraph and uses deepseek as brain of the bot

## Pre-requesite

- The project is build using python 3.11

## Cloning Project

```bash
# clone repo
git clone https://github.com/YaredAlex/erp-assistant.git
```

## Create environment

```bash
python -m venv env
#activate environment
# env/scripts/activate
```

## Installing dependancies

```bash
pip install -r requirements.txt
```

## Add neccessary Environment variable

- remember to edit those values

```bash
DEEPSEEK_API_KEY =YOUR_KEY
DB_USER =USER
DB_PASSWORD =PASSWORD
DB_BASE =YOUR_DATABASE
DB_HOST =HOST
```
